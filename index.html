Towers of Hanoi

The objective of the puzzle is to move the entire stack to another rod, obeying the following simple rules:

  1. Only one disk can be moved at a time.
  2. Each move consists of taking the upper disk from one of the stacks and placing it on top of another stack i.e. a disk can only be moved if it is the uppermost disk on a stack.
  3. No disk may be placed on top of a smaller disk.

Explanation

After you understand the rules, it is clear that to move the disk at the bottom of the stack to the destination tower, you must first move the n-1 disks from the first tower to the extra tower.

Thus, the steps to move n disks from the first tower to the destination tower are as follows:

  1. Move n-1 disks from tower 1 to the extra tower using the destination tower.
  2. Move the disk from tower 1 to the destination
  3. Move n-1 disks from the extra tower to the destination using tower 1.

Using these steps, we can translate that into a recursive algorithm. Here it is in JavaScript:

        
            function hanoi(nDisks, source, to, using){
                if( nDisks > 0 ){
                    hanoi(nDisks - 1, source, using, to); // step 1
                    console.log('move disk from ' + source + ' to ' + to); // step 2
                    hanoi(nDisks - 1, using, to, source); // step 3
                }
            }
